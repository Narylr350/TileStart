# TileStart

## Goal

开发一个兼容 Windows 10 和 Windows 11 的系统级磁贴开始菜单替代程序。界面和主要行为复刻用户当前的 Win10 扩展开始菜单，同时加入 Dawn Launcher 风格的任意项目固定能力。

安装后通过系统开始按钮和单独的 `Win` 键打开 TileStart，先在 Win10 上对照原生开始菜单验证，升级 Win11 后继续使用相同的磁贴体验。

## Users and Scenarios

主要用户是项目作者本人。

- 隐藏桌面图标，通过 `Win` 键或任务栏开始按钮打开 TileStart。
- 将开发工具、游戏、文件夹、文件、脚本和网址按磁贴分组管理。
- 从资源管理器直接拖入项目。
- 在 Win10 上与原生开始菜单比较视觉和交互，后续在 Win11 上保持相同使用习惯。

## MVP

- 底部锚定、不覆盖任务栏、尺寸可调且可记忆的 Win10 风格窗口，支持多显示器和 Per-Monitor DPI。
- 左侧扫描开始菜单与 UWP/MSIX 应用，提供最近添加和字母索引；用户直接输入搜索时转交 Windows Search，TileStart 不实现独立搜索结果页。
- 右侧提供 8 格分组，以及 `1×1`、`2×2`、`4×2`、`4×4` 四种磁贴尺寸，支持拖动、跨组移动和自动避让。
- 支持固定 `.exe`、`.lnk`、普通文件、文件夹、`.bat`、`.cmd`、`.ps1`、URL、UWP/MSIX 和自定义命令。
- 支持名称、图标、参数、工作目录、管理员运行、磁贴尺寸和取消固定等设置。
- 提供安装程序、用户登录自启动、托盘、单实例、单独 `Win` 键接管和任务栏开始按钮接管。
- Explorer 重启后自动恢复接管；Host 不可用时放行原生开始菜单；托盘可暂停接管并打开原生开始菜单。
- 将窗口、布局、目标和图标配置保存在本地，重启后恢复。

## Inputs and Outputs

### 输入

- 开始菜单快捷方式和 `shell:AppsFolder` 应用。
- 用户拖入的文件、文件夹、脚本和网址。
- 用户设置的图标、参数、布局和运行方式。
- `Win` 键、任务栏开始按钮和 Explorer 生命周期事件。

### 输出

- Win10 风格磁贴开始菜单。
- 通过 Windows Shell 启动目标。
- 本地布局、设置、图标缓存和诊断日志。
- Win10/Win11 x64 安装程序和便携测试版本。

## Non-goals

- 不替换整个 Windows Explorer，不重做完整任务栏。
- 不复刻 ExplorerPatcher 或 Dawn Launcher 的全部功能。
- 不读取 Windows 原生 Live Tile 数据，不做天气和新闻等在线磁贴。
- 不实现独立搜索结果页；开始菜单中的搜索输入转交 Windows Search。
- 不做云同步、账号、插件市场、自动更新和非 Windows 平台版本。

## Seed Tasks

1. 在当前 Win10 19045 上定位开始按钮与 `Win` 键触发开始菜单的调用路径，制作最小 C++ Hook 探针，验证点击、按键、Explorer 重启和失败放行。
2. 创建 WPF Host、C++ Shell Hook、Injector 和 Named Pipe IPC 的最小架构，确保 Host 未运行时原生开始菜单仍可打开。
3. 实现底部锚定窗口、左侧应用列表和 8 格 Win10 磁贴布局，并对照当前 150% DPI 截图调整尺寸。
4. 实现任意文件、文件夹、快捷方式、脚本和网址的拖入、解析、编辑、启动和配置保存。
5. 实现安装、自启动、托盘、Hook 生命周期管理、卸载恢复，并完成 Win10 实机回归。
